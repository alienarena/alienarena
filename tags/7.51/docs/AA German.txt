1.	EINFÜHRUNG

ALIEN ARENA ist ein eigenständiger 3D first person online deathmatch shooter 
basierend auf den originalen source codes von Quake II und Quake III, welche von ID Software
unter der GPL Lizenz veröffentlicht wurden. Mit Features wie 32 Bit Grafiken, 
neuer Partikel Engine und Effekten, Licht Blooms, reflektierendes Wasser, hoch auflösende Texturen und skins,
hi poly Modellen, stain maps, ALIEN ARENA bringt die Grafik auf die gleiche Ebene, wie die heutigen Top Spiele.

Das Spiel enthält 37 Level, welche online gegen andere Spieler, 
oder gegen die eingebauten CodeRED Bots gespielt werden können.

Alien Arena bietet CTF, AOA (All Out Assault - Modus, in der Spieler mit Fahrzeugen in die Schlacht ziehen können), 
Deathball, und Team Core Assault. Gehen Sie einfach in das Multiplayer-Menü und starten einen Server, 
stellen die gewünschten Spielregeln ein, wählen eine Map aus, auf der sie spielen wollen.
Sie auch zwischen fünf verschiedenen Mutatoren wählen (instagib, rocket arena, excessive, low grav, regeneration, un vampire), 
um das Spielerlebnis zu erweitern.

3.	START

*	INSTALLATION

(Windows)

Die Installation is sehr einfach. Nach dem Beende des Downloads, einfach die alienarena2008.exe Datei starten 
und die Installation beginnt. Folgen Sie den Anweisungen des Installshield Setup Programms.
Wenn Sie Windows 2000 oder XP benutzen, müssen Sie über Administrator Rechte verfügen. 
Sollten Sie nicht die erwähnten Rechte besitzen, kann es passieren, dass Sie eine Fehlermeldung erhalten, 
wenn das Programm versucht die Uninstall Informationen in die Registrierung zu schreiben. 
Wenn das bei Ihnen der Fall ist, drücken Sie einfach auf "ignore" und die Installation wird ohne probleme fortgesetzt.
Nachdem die installation abgeschlossen ist, können Sie entweder das Alien Arena Symbol, was sich auf Ihrem Desktop befindet,
anwählen um das Spiel, oder aber das Alien Arena Server Symbol um einen Dedicated Server zu starten.

(Linux)

Entpacken Sie einfach das Archiv entweder in Ihren /usr/local/games Ordner oder in einen Ordner nach Ihrer Wahl.
Tippen Sie ./crx um das Spiel, oder ./crded um eien Dedicated Server zu starten. 
Alternativ können Sie auch die Schnellstart Symbole, die im Menü installiert sind verwenden.
Die Source dateien sind enthalten, damit Sie die Binär Daten selbst kompilieren, falls benötigt.
Bitte besuchen Sie unser Forum, wenn Sie irgendwelche Fragen dazu haben.

* Grundlegende Befehle

ALIEN ARENA wird fast genauso wie Quake und Unreal Tournament gespielt, bis auf einige Ausnahmen.
Der Spieler kann folgende Waffen benutzen:

1.	Blaster
2.	Alien Disruptor
3.	Chaingun
4.	Flame Thrower
5.	Rocket Launcher
6.	Alien Smart Gun
7.	Alien Beamgun
8.	Alien Vaporizer
9.	Violator

Alle Waffen haben einen alternativen Feuer Modus. In Ihrem Tastenbelegungs Menü, 
wählen Sie Ihre Taste für den sekundären Feuer Modus aus, damit Sie diesen auch benutzen können.
Es sind neuer Powerups vohanden, wie Haste (Geschwindigkeit) und Sproing (Sprung), 
oder auch der ehrwürdige Quad Damage (Vierfach Schaden) und Invulnerability (Unverwundbarkeit).
Als Spieler verfügen Sie außerdem noch über eine Taschenlampe, welche Sie von Anfang an dabei haben und sehr nützlich ist. 
Mit der "F" Taste wird sie ein- und ausgeschaltet, oder Sie können dafür eine beliebige Taste einstellen.
Es ist ebenfalls ratsam, die Grafikeinstellungen Ihren Bedürfnisse und Systemanfoderungen anzupassen, 
um das beste Ergebnis zu erzielen.
Das Spiel ist zu Beginn auf einem Mittelwert voreingestellt, Sie sollten also, je nach Ihrer Frame Rate, 
rumprobieren und gegebenenfalls Effekte hinzufügen und entfernen. 
Um sich die Frame Rate während des des Spiels anzeigen zulassen, 
können Sie in der Konsole (wird mit der ^ Taste aufgerufen) "set cl_drawfps 1" eingeben.
ALIEN ARENA erlaubt es Ihnen auch farbige Namen wie in Quake 3 zu benutzen, dazu müssen Sie 
(in der Konsole) die Tasten Umschalt und 6(&) drücken, danach eine Nummer, um die Farbe einzustellen.
Das Movement (Bewegung) basiert auf der Quake 2 Physik, die Ihnen Starfejumping 
(springen bei gedrückter Seitenschritt Taste), Doublejumping (doppelter Sprung) 
und andere für Quake typische Bewegungen erlaubt.
In Alien Arena können Sie auch Dodgen (seitwärts Sprung), ähnlich wie in Unreal Tournament, 
indem Sie die Strafe (Seitenschritt) Taste drücken, danach komplett loslassen und dieselbe taste nochmals drücken.

* LEVEL BEENDEN

In Alien Arena können Sie ein Einzelspieler Turnier spielen, indem Ihr Ziel ist, 
das Fraglimit (Obergrenze der Abschüsse) zu erreichen, bevor es ein Bot (Computergegner) schafft. 
Sollten Sie es nicht shaffen, werden Sie gezwungen den Level zu wiederholen, bis es shaffen. 
Diese Regeln können Sie im Menü unter System ändern.
Sie können ebenfals gegen menschliche Gegner online spielen, Ihr Ziel ist es das Fraglimit vor Ihren Gegnern, 
oder die meisten Frags (Abschüsse) zu erreichen, bevor das Zeitlimit überschritten ist.

Im CTF Modus, sind Sie in einem Team, versuchen die gegnerische Flagge zu stehlen und sie zum Standpunkt 
Ihrer eigenen Flagge zu bringen. Das Team, welches das Punkte Limit erreicht, gewinnt.

Beim Deathball ist das Ziel, so viele Punkte wie möglich zu machen, entweder durch das Töten von Gegnern, 
oder den Ball in das gegnerische Tor zu schiessen (was Ihnen 10 Punkte einbringt). 
Wenn sich ein Spieler innerhalb des Deathball befindet, ist er wehrlos.

Beim Team Core Assault, ist Ihr Ziel, die gegnerischen Power Nodes zu deaktivieren und danach die Spider Node zu zerstören.
Sie können der Spider Node nur Schaden zufügen, wenn alle anderen Power Nodes deaktiviert sind.

* ZU EINEM SERVER VERBINDEN

Sie können entweder "join server" in dem Multiplayer Menü und einen Server in der Liste auswählen, 
oder Sie können den CodeRED Galaxy Server Browser (nur unter Windows) benutzen. Im In- Game Browser, 
sind die Server nach dem Ping sortiert, von oben mit dem niedrigsten beginnend.

4.	THE CODERED ACE BOT

Die Bots sind in ALIEN ARENA eingebaute Computergegner. Mehrere von ihnen sind bereits für Multiplayer vorkonfiguriert.
Jeder Level in Alien Arena, hat eine bestimmte Bot Datei, das vorgibt welche Bots in welchen Levels spielen.

Um einen Bot hinzuzufügen, geben Sie (in der Konsole) "sv addbot name model/skin" ein 
und um ihn wieder zu entfernen "sv removebot name".
Sie können Bots auch im Menü unter Deathmatch/Bots Flags hinzufügen.

In den Deathmatch Optionen können Sie auch noch andere Einstellungen wie Chatten, 
Node Speicherung und Zielgenauigkeit vornehmen.

Die Bots können Sie auch komplett selber mit dem Botconfigurator Programm einstellen. 
Sie können die Geschicklichkeitsstufen (skill levels), Genauigkeit (accuracy), bevorzugte Waffen (weapon favoring), 
Bewusstsein (awareness) und die Sätze, die dann auf dem Bildschirm erscheinen (chat strings) bestimmen.

Bots mit der Geschicklichkeitsstufe 0 sind einfach zu besiegen. Auf Stufe 1 sind sie ein bißchen schwerer, 
dodgen mehr und schiessen genauer. Auf Stufe 2 dodgen sie etwas besser, 
benutzen Rocket Jumps und sind noch genauer beim Schiessen.

Bots auf Stufe 3 sind extrem geschickt, Strafen, benutzen Strafe Jumping um durch den Level zu kommen.

Wenn die Bots Sie töten, werden Sie hinterher von ihnen verspottet (taunt). 
Wenn Sie Alien Arena im Singleplayer Turnier Modus gegen Bots spielen, 
wird das Auswählen von "easy" alle Bots auf Stufe 0 stellen.
Durch das Auswählen von "normal" werden die Bots auf die von Ihnen eingestellte Stufe konfiguriert. 
Das Auswählen von "hard" stellt alle Bots um eine Stufe höher.

5.	DEINSTALLATION

Um Alien Arena zu deinstallieren, gehen Sie einfach in Ihr Start Menü, wählen den Alien Arena Ordner 
aus und klicken auf "Uninstall".

Linux Benutzer müssen einfach Ihren Alien Arena Ordner löschen.
